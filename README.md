### bt-express
基于Express实现的一套轻量级的MVC框架
### 目录结构

```
├── app                         #用户应用目录
│   ├── config                 
│   │   ├── config.js           #用户配置文件
│   ├── controllers             #控制层目录
│   ├── modules                 #数据层目录
│   ├── views                   #视图层目录
├── public                      #资源目录,scss,js,jpg
└── system                      #bt-express框架目录
```
    

### 特性
=====

1. OOP的编码方式
2. MVC代码风格
3. 自定义DB配置选项
4. 可配置controllers(路由)多层嵌套
5. 配置的MVC目录名称
6. 独立封装Mysql操作类
7. Query支持链式查询
8. 入全局utils工具
